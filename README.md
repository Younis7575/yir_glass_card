# 🧊 Yir Glass Card

A beautiful **glassmorphism card widget** for Flutter with custom shape, blur, gradient, shadow and child support.

---

## ✨ Features

- 🧊 Glass blur effect
- 🎨 Custom color support
- 🌈 Gradient background
- 🌑 Shadow control
- 📐 Custom shape (your design)
- 👶 Child widget support

---

## 📦 Installation

```yaml
dependencies:
  yir_glass_card: ^0.0.1
